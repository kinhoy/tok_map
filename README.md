# tok_map
The map of The Legend of Zelda: Tears of the Kingdom

**免责声明，本地图项目代码仅用于学习和辅助玩家个人游玩，地图资源来自网络，请勿用于商业用途。**

塞尔达传说：王国之泪的地图，从网上的资源来看，由天空，陆地和地底三张地图构成，因为特别喜欢前作，也购买了新作，便有了这个项目，由于游戏尚未解禁，本地图项目并无任何标注点，敬请期待后续游戏正式开放游玩！！！
